# Fitness Frontend

React frontend for the Fitness Tracker app.

## Features

- Meal and macro tracking by date
- Food barcode lookup and camera scanning
- Workout exercise logging
- Account login, registration, and local profile support
- User settings

## Scripts

```powershell
npm start
npm test -- --watchAll=false
npm run build
```

The app expects the backend API at `http://localhost:5000`.
